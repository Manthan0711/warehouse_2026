🔄 Key Data Flows in System:
1. Owner Creates Listing:
Owner Dashboard → Form Input → Warehouse Service → PostgreSQL Insert → RLS Check → Image Upload → Storage
2. Seeker LLM Search:
Search Query → OpenAI Intent Extraction → Search Service → PostgreSQL Query (RLS Applied) → Results with Images → Frontend Display
3. Booking Request:
Seeker Select Dates → Booking Service → Check Availability → Insert to DB → Real-time Notification → Owner Approval
4. Review & Sentiment:
Seeker Review → OpenAI Sentiment Analysis → Profile Service → Store in DB → Calculate Rating → Display
Authentication Flow:
Login → Supabase Auth → JWT Token Generated → Stored in Browser → All Requests Include JWT → Backend Verifies Token → RLS Policies Enforced
