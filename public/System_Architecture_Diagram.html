⚙️ ML Engine — 12 Algorithms (2,142 LOC)
  • 📦
    PRIMARY: 5-Algorithm Ensemble (ml-algorithms.ts — 1,563 LOC)
    Default for all /api/recommend requests
  • 1️⃣
    KNN — K=8, 8-dim weighted Euclidean, location weight=6.0
  • 2️⃣
    Random Forest — 50 trees, 80% bootstrap, √n feature bagging
  • 3️⃣
    Gradient Boosting — 10 iterations, η=0.1, decision stumps
  • 4️⃣
    Neural Network — 10→10→4→1 MLP, Leaky ReLU
  • 5️⃣
    Hybrid Stacking — Meta-learner over KNN + RF
  • 📦
    SECONDARY: 5-Algorithm Advanced (advanced-ml-algorithms.ts — 579 LOC)
  • 6️⃣
    KNN Gaussian Kernel — K=15, σ=0.8, 6-dim vector
  • 7️⃣
    Content-Based Filtering — TF-IDF-like, location=0.35
  • 8️⃣
    Collaborative Filtering — Simulated user patterns
  • 9️⃣
    Sigmoid Neural Network — 6→3→2→1 MLP, sigmoid
  • 🔟
    Bayesian Scoring — Prior × Likelihood / Evidence
  • 🎯
    +2 Support Algorithms: License Expiry Validator, Dynamic Weight Calculator
📄 Document Verification Pipeline (552 LOC)
  • 🧠
    TensorFlow.js + MobileNet v2
    Neural network document vs non-document classification
  • 🔤
    Tesseract.js OCR — Text extraction from PDF (up to 5 pages) and images
  • 📏
    Levenshtein Distance — Strict 80% threshold for credential matching
  • 🔍
    Regex Pattern Detection — GST: /[0-9]{2}[A-Z]{5}[0-9]{4}[A-Z]Z[0-9A-Z]/ | PAN: /[A-Z]{5}[0-9]{4}[A-Z]/
  • 📊
    100-Point Scoring System:
    GST(10+15) + PAN(10+15) + Name(15) + Company(15) + Phone(10) + DocType(10)
  • ✅
    Decision: ≥70 → APPROVE | ≥40 → REVIEW | <40 → REJECT
